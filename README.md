# apache-spark-reels

Content (narration `tts/` + generated `audio/`) for the **Apache Spark** topic in
[graphl-mobile](../graphl-mobile), the technical reels PWA. The app fetches the
`audio/<stem>.wav` at runtime via raw GitHub; it is not bundled into the app.

See **[CLAUDE.md](./CLAUDE.md)** for structure, the authoring/audio pipeline,
series conventions, and the scene index.
